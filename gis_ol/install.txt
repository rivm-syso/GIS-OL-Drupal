Installatie van de gis_ol module:
1. Plaats de bestanden in de folder: ....drupal\htdocs\sites\all\modules\gis_ol
2. Installeer de module gis_ol via de admin-extend functionaliteit.

Update van de gis_ol module:
1. Lees in deze file welke fields er extra moeten worden aangemaakt. Dit moet handmatig gebeuren via een edit van de fields van het content-type gis_ol. Het betreft:
	Versie	Field						Parameters
	   1.0	gis_ol_download				
2. Overschijf de bestanden in de folder: ....drupal\htdocs\sites\all\modules\gis_ol
3. Clear de cache.

Verwijderen van de gis_ol module (t.b.v. ontwikkeling):
1. Installeer drupal helemaal opnieuw.
